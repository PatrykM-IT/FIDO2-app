# Magisterka_FIDO2
Tworzenie demonstratora w postaci aplikacji webowej, która wykorzystuje technologię protokołu FIDO2 oraz fizyczne klucze.

App build with this tut: https://www.youtube.com/watch?v=-RCnNyD0L-s

Used depends:

node.js
nodemon
ejs
express
express-session
bcrypt
mysql2
dotenv
passport 
passport-local 
express-flash
method-override
